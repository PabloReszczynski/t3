## Instrucciones para correr el programa

### Requerimientos:
Node 6.2.2 con npm

Para instalar las dependencias debes correr:
```bash
npm install
```

Para correr los tests:
```bash
npm test
```

Para ejecutar un programa:
```bash
node main.js "{+ 1 2}"
> 3
```
o bien, ejecutando sin un argumento lleva a un prompt del interprete.
